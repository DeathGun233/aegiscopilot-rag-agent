# AegisCopilot 面试指南

## 项目定位

AegisCopilot 是一个面向面试的企业知识库 AI Agent 项目。它结合了基于依据的问答、任务式工作流路由、本地持久化、离线评估和仪表板前端。

## 演示亮点

- 带有引用的企业级依据问答
- 面向任务的工作流追踪
- 持久的文档、对话、任务和评估报告
- 可配置的生成提供者：`mock` 或 `openai-compatible`
- 检索预览和系统统计用于可观测性

## 建议演示流程

1. 打开仪表板并解释系统统计卡片。
2. 提出一个知识性问题，例如 `员工请假需要提前多久申请？`
3. 提出一个任务性问题，例如 `请总结差旅报销流程`
4. 展示 `Agent Trace`、检索预览和引用卡片。
5. 运行离线评估并解释指标。
6. 最后描述如何升级到向量数据库、重排序器和 LangGraph。

## 架构要点说明

- 对话层用于历史记录和多轮上下文
- Agent 层用于意图检测、计划选择、工具使用和依据检查
- RAG 层用于分块、检索、支持重排序的接口和引用
- 平台层用于持久化、指标、评估和部署

## 工程要点强调

- 提供者抽象使生成逻辑可互换
- 持久化让项目感觉是可运营的而非玩具级
- 检索预览使调试和演示更容易
- 评估增加了可衡量的优化循环

## 发展路径

- 用嵌入 + 混合搜索替换轻量级检索
- 添加 Cross-Encoder 重排序
- 将持久化迁移到 PostgreSQL、Redis 和 pgvector
- 用手写工作流替换为 LangGraph
