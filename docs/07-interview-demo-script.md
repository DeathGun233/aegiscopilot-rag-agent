# 面试演示脚本

## 3 到 5 分钟版本

### 1. 项目定位

这是一个企业知识库 AIAgent，不是普通聊天机器人。重点不是自由生成，而是基于企业内部文档做可溯源回答，并且能处理任务型问题。

### 2. 核心架构

- 对话层负责会话与上下文
- Agent 层负责意图识别、路径规划、工具调用和 grounded output
- RAG 层负责切分、检索和引用
- 平台层负责评估、持久化、配置和演示能力

### 3. 我做的工程化点

- 做了本地持久化，让文档、会话、任务和评估结果都能保留
- 做了 provider abstraction，可以从 mock 生成切换到兼容 OpenAI 的接口
- 做了检索预览和系统指标页，让回答过程可观察
- 做了离线评估，能量化回答率、引用命中率和关键词命中率

### 4. 我会怎么继续升级

- 把轻量检索升级为 embedding + hybrid retrieval
- 加入重排模型
- 把本地存储升级成 PostgreSQL / Redis / pgvector
- 用 LangGraph 替换当前 workflow 编排
