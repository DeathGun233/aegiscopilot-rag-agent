# 项目背景与目标

## 为什么做企业知识库 Agent

企业内部已经沉淀了大量制度、流程、研发规范、产品文档和 FAQ，但这些知识通常分散在 Wiki、网盘、IM 群和邮件中。普通聊天机器人虽然能对话，但并不能保证回答来自企业真实资料，也缺乏对任务流程的理解。

AegisCopilot 的目标是把“知识检索、对话管理、任务规划、引用溯源、效果评估”整合成一个可学习、可扩展、可面试讲解的完整项目。

## 解决的问题

- 降低员工查找制度和流程的时间成本
- 让复杂问题从“只回答一句话”升级到“检索 + 分析 + 汇总”
- 通过引用和 grounded check 降低幻觉风险
- 为后续接入多模态、工具系统、微调和线上部署打基础

## 与普通聊天机器人的区别

普通聊天机器人偏向开放式生成；企业知识库 Agent 更强调：

- 回答必须能溯源
- 无证据时要拒答或保守回答
- 需要会话管理与任务拆解
- 需要工程化评估和部署能力
